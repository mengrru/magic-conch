# Surf

## Usage

Write JSON into file `index.json` followed format below:

```
{
  "name what you want": {
    "name": "name what you want, it will be shown as option name",
    "items": [
      "https://sadgrl.online/",
      "https://novov.me/",
      ...
    ]
  },
  ...
}
```

Then visit `index.html` you can start surfing into your collections randomly!
